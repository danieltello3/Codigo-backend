django-admin startproject <project_name> => crea un proyecto con ese nombre
python manage.py runserver => levanta el servidor
python manage.py showmigrations => lista todas las migraciones que hay en el proyecto y las que han sido efectuadas estaran marcadas con una X
python manage.py migrate => efectua todas las migraciones que aun no se han realizado en la bd
python manage.py startapp libreria => crea una aplicacion para que podamos crear modelos, vistas, rutas, entre otros
